## God Mode {{ date: '2024-01-26T11:21:32Z' }}


We now have a God Mode(<kbd>Cmd</kbd> + <kbd>K</kbd>)!
You can query your Sqlite databases with SQL and chat 
with it using plain English. It knows your database schema, indexes, triggers and other metadata in order to give you the most accurate
and relevant answers. 

It uses OpenAI ChatGPT so you can just bring your own API Key to get started&mdash;your API key and other info won't leave your computer/server as it's saved locally.
